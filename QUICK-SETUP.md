# ⚡ راهنمای سریع تنظیم JSONBin.io (2 دقیقه)

## مشکل فعلی
داده‌ها فقط در localStorage مرورگر ذخیره می‌شوند و بین دستگاه‌ها (گوشی و دسکتاپ) همگام نمی‌شوند.

## راه حل: تنظیم JSONBin.io

### مرحله 1: ساخت حساب (30 ثانیه)
1. به **https://jsonbin.io** بروید
2. روی **"Sign Up"** کلیک کنید
3. با ایمیل یا GitHub ثبت‌نام کنید

### مرحله 2: دریافت API Key (30 ثانیه)
1. بعد از ورود، به **Dashboard** بروید
2. در بخش **"API Keys"**، روی **"Create API Key"** کلیک کنید
3. یک نام انتخاب کنید (مثلاً: `atiyeh-form`)
4. **API Key** را کپی کنید

### مرحله 3: تنظیم در پروژه (1 دقیقه)
1. فایل `config.js` را باز کنید
2. API Key را در این قسمت وارد کنید:

```javascript
const JSONBIN_API_KEY = 'API_KEY_شما_اینجا'; // API Key خود را اینجا وارد کنید
```

3. فایل را ذخیره کنید
4. دوباره deploy کنید:

```bash
vercel --prod --yes
```
